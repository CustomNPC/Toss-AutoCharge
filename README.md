# Usage
POST http://127.0.0.1:8080/check<br/>
Content-Type JSON<br/>
Body
```js
{
    "tossId": "Toss-ID HERE",
    "name": "senderName",
    "amount": Number
}
```

Response:
```js
{
    "success": Boolean,
    "found": Boolean,
    "message": "an error message if success is false"
}
```
